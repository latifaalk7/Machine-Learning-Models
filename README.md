# Annotated ML Concepts
Regression, Classification, Tree-based Ensembles, Unsupervised Learning, Model Evaluation

## 📁 How the Folders are Organized

*   **Regression**: Projects that predict a specific number.
*   **Classification**: Projects that sort data into categories.
*   **Tree-Based-Ensembles**: Advanced models that combine multiple "decision trees" together to make highly accurate predictions (like Random Forest and XGBoost).
*   **Unsupervised-Learning**: Projects that look for hidden patterns in data without using ready-made labels.
*   **Model-Evaluation**: Code used to test the models properly and make sure their accuracy scores are reliable.

## 🛠️ Tools Used

*   **Language**: Python
*   **Main Libraries**: NumPy, Pandas, Scikit-Learn, XGBoost
*   **Visuals**: Matplotlib, Seaborn
